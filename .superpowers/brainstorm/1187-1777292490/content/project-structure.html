项目目录结构

static/ 是项目根目录下的文件夹

d:\桌面\agent\
├── .venv/                ← Python 虚拟环境
├── static/
│   └── uploads/          ← 📁 上传的图片存这里
│       ├── 20260427_001.jpg
│       ├── 20260427_002.jpg
│       └── ...
├── templates/           ← Jinja2 页面模板
│   ├── index.html         ← 帖子管理首页
│   ├── editor.html        ← 新建/编辑帖子
│   └── history.html       ← 发布历史
├── web/                ← FastAPI 路由
│   └── routes.py
├── agent/              ← LangChain Agent
│   └── poster_agent.py
├── scheduler/          ← APScheduler 调度
│   └── scheduler.py
├── publisher/          ← 发布接口（可插拔）
│   └── publisher.py
├── models/             ← 数据库模型
│   └── models.py
├── main.py                ← FastAPI 启动入口
└── config.py              ← 配置文件

路径对应关系

  • 文件系统: d:\桌面\agent\static\uploads\xxx.jpg
  • 数据库记录: /static/uploads/xxx.jpg
  • 页面访问: <img src="/static/uploads/xxx.jpg">
  • FastAPI 挂载: app.mount("/static", StaticFiles(directory="static"))
